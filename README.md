# 🎮 Squid Game Twitter Analysis Dashboard (Tableau)

## 📌 Project Overview
<img width="1905" height="986" alt="image" src="https://github.com/user-attachments/assets/af01a640-7fe3-4e96-94ef-c3d57fe0004c" />

This project analyzes Twitter data related to the popular Netflix series **Squid Game** using **Tableau**.
The dashboard explores tweet activity, user engagement, tweet length distribution, and platform sources to understand how people interacted with the show on Twitter.
The goal of this project is to demonstrate **data visualization, social media analytics, and dashboard design skills using Tableau.**

---

# 📊 Dashboard Insights

The dashboard provides several key insights:

### 1️⃣ Total Users

* Total Twitter users in the dataset: **52,240**

### 2️⃣ Total Tweets

* Total tweets related to Squid Game: **75,456**

This indicates strong engagement and discussion about the show.

---

# 📈 Key Visualizations

## Tweet Length Distribution

Shows how long tweets are when users discuss Squid Game.

Insights:

* Most tweets contain **short text**
* Users prefer quick reactions and short opinions
* Very long tweets are less common

---

## Tweets Over Time

This line chart shows how tweet activity changed over time.

Insights:

* Tweet activity **spiked when the show became popular**
* Activity decreases after the peak period
* Social media conversations follow trending events

---

## Top 10 Users by Followers

Displays the most influential Twitter accounts discussing Squid Game.

Insights:

* Major **news channels and media pages** dominate the conversation
* High follower accounts help amplify the trend

---

## Total New Users by Year

This bubble chart shows when users created their Twitter accounts.

Insights:

* Large number of users joined Twitter between **2019–2021**
* This reflects recent growth in social media adoption

---

## Top Tweet Sources

Shows which platforms users used to tweet.

Insights:

* **Twitter for iPhone** generated the highest tweets
* Followed by **Twitter for Android**
* Web App and TweetDeck have smaller usage

---

# 🎛 Dashboard Filters

The dashboard includes interactive filters:

These filters allow users to explore different segments of the data.
